# @luvano/prettier

Enterprise-grade, extensible Prettier configuration used across all Luvano Team repositories.

## ✨ Features

- Modular presets (base, TypeScript, React, Nest, Markdown)
- Auto-applied overrides (JSON, YAML, GraphQL)
- Support for plugins:
  - prettier-plugin-sort-imports
  - prettier-plugin-tailwindcss
- Deep configuration merging
- Schema-validated custom overrides
- Zero-config usage — but fully extensible
